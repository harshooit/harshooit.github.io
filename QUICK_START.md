# 🚀 Quick Start Guide - DevShell OS

Welcome to DevShell OS! This guide will help you get started quickly.

## 🎬 First Time Setup

1. **Power On**: Click the "⏻ Power On" button
2. **Watch Boot**: Enjoy the authentic boot sequence
3. **Explore Desktop**: You're now on the desktop!

## 🖱️ Basic Controls

### Desktop
- **Single Click**: Select an icon
- **Double Click**: Open an icon/window
- **Right Click**: Open context menu
- **Click & Drag**: Select multiple icons
- **Click Empty Space**: Create ripple effect

### Windows
- **Drag Title Bar**: Move window
- **Drag Bottom-Right**: Resize window
- **Drag to Edge**: Snap window (left/right/maximize)
- **Click Title Bar Buttons**:
  - `_` = Minimize
  - `□` = Maximize/Restore
  - `✕` = Close

### Taskbar
- **Start Button**: Open Start menu
- **Window Buttons**: Switch between windows
- **System Tray Icons**:
  - 🎨 = Change theme
  - ⚡ = Performance monitor
  - 🔍 = Search
- **Clock**: Shows time and date

## ⌨️ Keyboard Shortcuts

### Essential
- `1` - Open About Me
- `2` - Open Projects
- `3` - Open Favourites
- `4` - Open Contact
- `Ctrl+K` - Open Search
- `Ctrl+M` - Toggle Performance Monitor
- `Ctrl+T` - Change Theme
- `Ctrl+W` - Close Active Window
- `Esc` - Close Menus/Dialogs

### Advanced
- `F5` - Refresh Window
- `F11` - Fullscreen (browser)
- Type `sudo` - Terminal Easter Egg
- `↑↑↓↓←→←→BA` - Konami Code

## 🔍 Search Feature

1. Press `Ctrl+K` or click 🔍 in tray
2. Type your search query
3. Results appear instantly
4. Click any result to navigate
5. Press `Esc` to close

**Searches across:**
- Projects
- Skills
- Contacts
- Hobbies

## 🎨 Themes

**4 Color Themes Available:**
1. **Default** - Green (Hacker style)
2. **Cyan** - Blue tones
3. **Purple** - Purple hues
4. **Amber** - Orange/yellow

**Change Theme:**
- Press `Ctrl+T`
- Click 🎨 in system tray
- Use Start menu option

## ⚡ Performance Monitor

**Toggle:** Press `Ctrl+M` or click ⚡ in tray

**Shows:**
- FPS (Frames per second)
- Memory usage
- Uptime
- Active windows count

## 🎵 Music Control

- **Vinyl Disc** (bottom-right corner)
- Click to mute/unmute
- Spins when playing
- Hover for glow effect

## 🎮 Easter Eggs

### 1. Terminal Flash
- Type `sudo` anywhere on desktop
- Watch the terminal flash
- Get "access granted" message

### 2. Konami Code
- Enter: `↑↑↓↓←→←→BA`
- Windows will dance
- Themes will cycle rapidly
- Special notification appears

### 3. Window Shake
- Try closing windows
- 10% chance of shake effect
- Playful warning message

## 💡 Pro Tips

### Window Management
- **Snap Left**: Drag to left edge
- **Snap Right**: Drag to right edge
- **Maximize**: Drag to top edge
- **Restore**: Double-click title bar

### Multi-Select Icons
1. Click and hold on empty desktop
2. Drag to create selection box
3. Icons inside box get selected
4. Release to finish selection

### Taskbar Tricks
- Click window button to minimize/restore
- Active windows have colored button
- Buttons bounce when window gets focus

### Notification Management
- Notifications auto-dismiss after 4 seconds
- Click `✕` to close manually
- Multiple notifications stack vertically

## 🐛 Troubleshooting

### Music Not Playing?
- Click anywhere on desktop first
- Browsers require user interaction
- Check vinyl disc (should be spinning)

### Scrolling Not Working?
- Make sure window is focused
- Try clicking inside window first
- Use mouse wheel or scrollbar

### Window Won't Move?
- Click and hold title bar
- Don't click on buttons
- Drag to move

### Search Not Finding Results?
- Check spelling
- Try different keywords
- Search is case-insensitive
