## 🛠️ Built With

This website is built entirely using clean, modern, and responsive web technologies:

* **HTML5** – Structured, semantic markup for web accessibility.
* **CSS3** – Custom styling, layouts, and responsive media queries.
* **JavaScript (ES6+)** – Interactive elements and dynamic frontend behavior.

---

## 📂 Repository Structure

The project is organized cleanly into the following directories:

* `css/` – Contains stylesheets for layout and design.
* `js/` – Holds JavaScript files handling site interactivity.
* `media/` – Stores images, icons, and asset visuals.
* `index.html` – The main entry point of the application.
